Hexo3+Github搭建免费博客（三）评论/分享/统计

2015-07-15
Hexo

增加第三方评论工具

友言

据说多说问题很多，从hexo2升级到hexo3无法同步评论，界面也不好看，所以选了友言。

在_config.yml里添加：

#友言
uyan:
  enable: true

到友言网站注册账号，获取你的uid。把下面的代码复制粘贴到themes\*\layout\_partial\article.ejs

<% if (config.uyan.enable && page.comments){ %>

<section id="comment">
  <h1 class="title"><%= __('comment') %></h1>
  <div id="disqus_thread">
    <!-- UY BEGIN -->
    <div id="uyan_frame"></div>
    <script type="text/javascript" src="http://v2.uyan.cc/code/uyan.js?uid=你的uid"></script>
    <!-- UY END -->
  </div>
</section>

<% } %>

本地无法看到，上传到github就可以看了。

Disqus

到官网注册账号。右上角admin

填写信息

这里就是复制评论脚本的地方：

hexo已经帮写好Disqus代码，只需要在_config.yml填上以下代码。

# disqus
disqus_shortname: laker

进入profile，这个就是shortname

上传到github就可以看了：

加入右上角「fork me on github」

这里有 github 给出的各种样式，把代码插入到任意一个全局的模板文件中就行，比如layout.ejs的末尾。例如
